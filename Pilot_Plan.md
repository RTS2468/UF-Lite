# Pilot Plan (tiny, reversible)
- **Scope:** one topic, one subject, 10 claims, 5 promises, 5 actions.
- **Method:** extract → score facts (EG) → score actions (gates) → publish 1-page scorecard.
- **Reversibility:** start with public records only; no private data.
- **Risks & Mitigation:** bias/opacity (independent review), defamation (verifiable sources), consent (public data only).
